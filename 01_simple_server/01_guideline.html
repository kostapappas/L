📚 Τι θα χρειαστούμε - Βασικές Έννοιες

🐍 Python

Τι είναι: Μια γλώσσα προγραμματισμού που μοιάζει με απλά αγγλικά. Σαν να δίνετε οδηγίες στον υπολογιστή σας με απλές λέξεις!

Γιατί την χρησιμοποιούμε: Είναι η πιο εύκολη γλώσσα για αρχάριους και πολύ δημοφιλής για web development.

📦 Virtual Environment (venv)

Τι είναι: Φανταστείτε το σαν ένα ξεχωριστό "δωμάτιο" στον υπολογιστή σας όπου βάζετε μόνο τα εργαλεία για αυτό το project.

Γιατί το χρειαζόμαστε: Για να μην μπερδευτούν τα προγράμματα μεταξύ τους - κάθε project έχει το δικό του χώρο!

🌐 Flask

Τι είναι: Ένα εργαλείο που μετατρέπει τον υπολογιστή σας σε web server - σαν να γίνεται ο υπολογιστής σας μια μικρή ιστοσελίδα!

Γιατί το χρησιμοποιούμε: Είναι απλό και ιδανικό για μικρές ιστοσελίδες όπως αυτή που θα φτιάξουμε.

Οδηγίες για Windows

1 Εγκατάσταση Python

Πρώτα πρέπει να εγκαταστήσουμε την Python στον υπολογιστή μας.

  1. Ανοίξτε τον browser σας και πηγαίνετε στο: python.org/downloads
  2. Κάντε κλικ στο μεγάλο κίτρινο κουμπί "Download Python"
  3. Όταν κατέβει, κάντε διπλό κλικ στο αρχείο
  4. ΠΡΟΣΟΧΗ: ✅ Τσεκάρετε το κουτάκι "Add Python to PATH" στο κάτω μέρος!
  5. Κάντε κλικ "Install Now"
⚠️ Σημαντικό: Αν ξεχάσετε το "Add Python to PATH", θα έχουμε προβλήματα αργότερα!

2 Κατέβασμα του Project

Τώρα θα κατεβάσουμε τον κώδικα της ιστοσελίδας.

Ανοίξτε το PowerShell:

  1. Πατήστε το πλήκτρο Windows
  2. Πληκτρολογήστε "PowerShell"
  3. Κάντε κλικ στο "Windows PowerShell"

Τώρα πληκτρολογήστε τις παρακάτω εντολές μία-μία:

cd Desktop

Αυτό σας πάει στην Επιφάνεια Εργασίας

git clone [URL του repository από τον εκπαιδευτή]

Αυτό κατεβάζει τον κώδικα - ο εκπαιδευτής θα σας δώσει το URL

💡 Αν δεν έχετε git: Ο εκπαιδευτής θα σας δώσει έναν φάκελο σε USB ή θα κατεβάσετε ένα ZIP αρχείο

3 Εγκατάσταση του Project

Τώρα θα εγκαταστήσουμε όλα τα απαραίτητα για να τρέξει η ιστοσελίδα.

cd 01_simple_server

Μπαίνουμε στον φάκελο του project

python -m venv venv

Δημιουργούμε το "δωμάτιο" για το project μας

venv\Scripts\activate

Μπαίνουμε στο "δωμάτιο"

✅ Αν δείτε (venv) στην αρχή της γραμμής, τα πήγατε τέλεια! 
pip install -r requirements.txt

Εγκαθιστούμε το Flask

4 Εκκίνηση της Ιστοσελίδας!

Ήρθε η ώρα να δούμε την ιστοσελίδα μας!

python app.py
🎉 Συγχαρητήρια! Η ιστοσελίδα σας τρέχει!
Ανοίξτε τον browser και πηγαίνετε στο: http://localhost:5001

Οδηγίες για Mac

1 Έλεγχος Python

Τα Mac συνήθως έχουν ήδη Python. Ας το ελέγξουμε!

Ανοίξτε το Terminal:

  1. Πατήστε ⌘ + Space
  2. Πληκτρολογήστε "Terminal"
  3. Πατήστε Enter
python3 --version

Αν δείτε κάτι σαν "Python 3.x.x", είστε έτοιμοι! Αν όχι:

  1. Πηγαίνετε στο python.org/downloads
  2. Κατεβάστε την έκδοση για Mac
  3. Εγκαταστήστε την

2 Κατέβασμα του Project

cd Desktop
git clone [URL του repository από τον εκπαιδευτή]

3 Εγκατάσταση

cd 01_simple_server
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

4 Εκκίνηση!

python app.py
🎉 Ανοίξτε τον browser στο: http://localhost:5001

Οδηγίες για Linux

Οι οδηγίες είναι παρόμοιες με το Mac. Χρησιμοποιήστε τις εντολές από την καρτέλα Mac!

🎨 Ώρα για Δημιουργία! (15 λεπτά)

Τώρα θα προσαρμόσετε την ιστοσελίδα για τον δικό σας δήμο!

📝 Άνοιγμα στο Visual Studio Code

  1. Ανοίξτε το Visual Studio Code
  2. File → Open Folder → Επιλέξτε τον φάκελο "01_simple_server"
  3. Δείτε τα αρχεία στα αριστερά

✏️ Αλλαγές που θα κάνετε

1. Αλλάξτε το όνομα του Δήμου

Ανοίξτε το αρχείο templates/index.html

Βρείτε όλα τα "Δήμος Άρτας" και αλλάξτε τα με τον δικό σας δήμο

2. Προσθέστε τις δικές σας υπηρεσίες

Ανοίξτε το αρχείο app.py

Βρείτε τη λίστα services_list και προσθέστε υπηρεσίες του δήμου σας

3. Αλλάξτε τα στοιχεία επικοινωνίας

Στο app.py, βρείτε το contact_info και βάλτε τα στοιχεία του δήμου σας

4. (Προαιρετικό) Αλλάξτε τα χρώματα

Ανοίξτε το static/style.css και πειραματιστείτε με τα χρώματα!

💡 Tip: Με το GitHub Copilot, απλά γράψτε σχόλια για το τι θέλετε και θα σας προτείνει κώδικα!

🌐 Μοιραστείτε με τους Συμφοιτητές σας! (10 λεπτά)

🔍 Βρείτε την IP σας

Windows: Στο PowerShell γράψτε:

ipconfig

Ψάξτε για "IPv4 Address"

Mac/Linux: Στο Terminal γράψτε:

ifconfig | grep inet
Η IP σας θα είναι κάτι σαν: 192.168.1.XX ή 10.0.0.XX

👥 Πείτε στους άλλους

Δώστε στους συμφοιτητές σας τη διεύθυνση:

http://[Η-IP-ΣΑΣ]:5001

Π.χ. http://192.168.1.105:5001

⚠️ Πρέπει να είστε στο ίδιο WiFi δίκτυο!

🎉 Επισκεφτείτε τις σελίδες των άλλων!

Ανοίξτε τον browser και δοκιμάστε τις διευθύνσεις που σας δίνουν οι συμφοιτητές σας!

🔧 Συχνά Προβλήματα & Λύσεις

❌ "python" is not recognized (Windows)

Λύση: Δοκιμάστε py αντί για python

❌ Δεν μπορώ να δω τη σελίδα των άλλων

Λύσεις:

  • Ελέγξτε ότι είστε στο ίδιο WiFi
  • Ελέγξτε ότι το firewall επιτρέπει την πρόσβαση
  • Σιγουρευτείτε ότι ο server τρέχει (δεν σταμάτησε με Ctrl+C)

❌ ModuleNotFoundError: No module named 'flask'

Λύση: Ξεχάσατε να ενεργοποιήσετε το venv ή να κάνετε pip install
