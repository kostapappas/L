Ασκήσεις προγραμματισμού για κατανόηση Flask εφαρμογών

🎯 Στόχος των Ασκήσεων

Μέσα από αυτές τις ασκήσεις θα μάθετε:

  • Πως να τροποποιείτε Python μεταβλητές
  • Πως να προσθέτετε νέα πεδία σε φόρμες
  • Πως να υλοποιείτε πολύπλοκη λογική υπολογισμών
  • Πως να συνδέετε Python με HTML templates

⏱️ Συνολικός χρόνος: Περίπου 20 λεπτά για όλες τις ασκήσεις

🟢 Άσκηση 1: Αλλαγή Τιμών (Εύκολη - 5 λεπτά)

Επίπεδο: Αρχάριος

📋 Τι θα κάνετε:

Θα αλλάξετε τις τιμές του βασικού κόστους και του επείγοντος.

🔧 Οδηγίες βήμα-βήμα:

  1. Ανοίξτε το αρχείο app.py
  2. Βρείτε τις γραμμές με τις σταθερές (γύρω στη γραμμή 16-17):
    3. ΒΑΣΙΚΟ_ΚΟΣΤΟΣ = 3.0
ΚΟΣΤΟΣ_ΕΠΕΙΓΟΝ = 2.0
  3. Αλλάξτε τις τιμές σε:
    4. ΒΑΣΙΚΟ_ΚΟΣΤΟΣ = 4.0
ΚΟΣΤΟΣ_ΕΠΕΙΓΟΝ = 3.0
  4. Αποθηκεύστε το αρχείο
  5. Επανεκκινήστε τον server και δοκιμάστε τον υπολογιστή

✅ Αποτέλεσμα:

Τώρα το βασικό κόστος θα είναι €4.00 και το επείγον €7.00 συνολικά!

🟡 Άσκηση 2: Νέος Τύπος Πιστοποιητικού (Μέτρια - 7 λεπτά)

Επίπεδο: Ενδιάμεσος

📋 Τι θα κάνετε:

Θα προσθέσετε έναν νέο τύπο πιστοποιητικού στη λίστα επιλογών.

🔧 Οδηγίες βήμα-βήμα:

  1. Στο αρχείο app.py, βρείτε τη λίστα ΤΥΠΟΙ_ΠΙΣΤΟΠΟΙΗΤΙΚΩΝ (γύρω στη γραμμή 20-26)
  2. Προσθέστε έναν νέο τύπο, π.χ. "Φορολογικής Ενημερότητας":
    3. ΤΥΠΟΙ_ΠΙΣΤΟΠΟΙΗΤΙΚΩΝ = [
    'Γέννησης',
    'Θανάτου', 
    'Οικογενειακής Κατάστασης',
    'Σπουδών',
    'Στρατολογικής Κατάστασης',
    'Φορολογικής Ενημερότητας'  # ← Νέος τύπος
]
  3. Αποθηκεύστε και δοκιμάστε

✅ Αποτέλεσμα:

Θα δείτε τον νέο τύπο πιστοποιητικού στο dropdown menu!

🚀 Μπόνους:

Προσθέστε 2-3 ακόμη τύπους πιστοποιητικών της επιλογής σας!

🔴 Άσκηση 3: Έκπτωση για Συνταξιούχους (Δύσκολη - 8 λεπτά)

Επίπεδο: Προχωρημένος

📋 Τι θα κάνετε:

Θα προσθέσετε έκπτωση 50% για συνταξιούχους με νέο πεδίο στη φόρμα.

🔧 Οδηγίες βήμα-βήμα:

Μέρος Α: Αλλαγή στο app.py
  1. Στη συνάρτηση υπολογισμος_κοστους(), μετά τη γραμμή που παίρνει το "επειγον", προσθέστε:
    2. συνταξιουχος = request.form.get('συνταξιουχος') == 'ναι'
  2. Μετά τον υπολογισμό του επείγοντος, προσθέστε την έκπτωση:
    3. # Εφαρμογή έκπτωσης συνταξιούχου
if συνταξιουχος:
    συνολικο_κοστος = συνολικο_κοστος * 0.5  # 50% έκπτωση
    περιγραφη_υπολογισμου += " - Έκπτωση συνταξιούχου 50%"
  3. Στέλνετε τη μεταβλητή στο template:
    4. return render_template('αποτελεσμα.html',
                         ...
                         συνταξιουχος=συνταξιουχος)  # ← Προσθήκη
Μέρος Β: Αλλαγή στο calculator.html
  1. Μετά το πεδίο "Επείγον", προσθέστε νέο πεδίο:
    2. <!-- Συνταξιούχος -->
<div class="form-group">
    <label>👴 Συνταξιούχος:</label>
    <div class="radio-group">
        <label class="radio-label">
            <input type="radio" name="συνταξιουχος" value="οχι" checked>
            <span>Όχι</span>
        </label>
        <label class="radio-label">
            <input type="radio" name="συνταξιουχος" value="ναι">
            <span>Ναι - Έκπτωση 50%</span>
        </label>
    </div>
</div>
Μέρος Γ: Αλλαγή στο αποτελεσμα.html
  1. Προσθέστε νέα γραμμή στα στοιχεία αίτησης:
    2. <div class="detail-row">
    <span class="label">👴 Συνταξιούχος:</span>
    <span class="value">
        {% if συνταξιουχος %}
            <span class="badge discount">Ναι (Έκπτωση 50%)</span>
        {% else %}
            <span class="badge normal">Όχι</span>
        {% endif %}
    </span>
</div>

✅ Αποτέλεσμα:

Οι συνταξιούχοι θα πληρώνουν το μισό ποσό για όλα τα πιστοποιητικά!
