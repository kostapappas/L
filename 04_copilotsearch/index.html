⌨️ Essential VS Code Shortcuts

Τα βασικά shortcuts που θα σου αλλάξουν τη ζωή:

🔍 Navigation & Search

Command Palette - Πρόσβαση σε όλες τις εντολές Cmd + Shift + P
Quick Open - Άνοιγμα αρχείων γρήγορα Cmd + P
Go to Symbol - Εύρεση functions, classes στο αρχείο Cmd + Shift + O
Go to Line - Μετάβαση σε συγκεκριμένη γραμμή Ctrl + G
Find & Replace - Εύρεση και αντικατάσταση κειμένου Cmd + F
Find in Files - Αναζήτηση σε όλα τα αρχεία Cmd + Shift + F

✏️ Editing Magic

Multiple Cursors - Επεξεργασία πολλών σημείων ταυτόχρονα Alt + Click
Select Next Occurrence - Επιλογή επόμενης εμφάνισης λέξης Cmd + D
Select All Occurrences - Επιλογή όλων των εμφανίσεων Cmd + Shift + L
Move Line Up/Down - Μετακίνηση γραμμής πάνω/κάτω Alt + ↑/↓
Copy Line Up/Down - Αντιγραφή γραμμής πάνω/κάτω Shift + Alt + ↑/↓
Comment Toggle - Προσθήκη/αφαίρεση σχολίων Cmd + /

🎯 Layout & Views

Toggle Sidebar - Εμφάνιση/απόκρυψη sidebar Cmd + B
Toggle Terminal - Εμφάνιση/απόκρυψη terminal Ctrl + `
Split Editor - Διαίρεση του editor σε παράθυρα Cmd + \
Zen Mode - Πλήρης εστίαση στον κώδικα Cmd + K Z

🔧 Code Intelligence

Go to Definition - Πήγαινε στον ορισμό F12
Peek Definition - Γρήγορη προεπισκόπηση ορισμού Alt + F12
Find All References - Εύρεση όλων των χρήσεων Shift + F12
Rename Symbol - Μετονομασία σε όλο το project F2
IntelliSense - Εμφάνιση προτάσεων Ctrl + Space

🤖 GitHub Copilot Best Practices

💡 Γεγονός: Developers με Copilot είναι 55% πιο γρήγοροι και 88% πιο παραγωγικοί!

🎯 Τι κάνει ΤΈΛΕΙΑ το Copilot:

  • Boilerplate Code: Γρήγορη δημιουργία βασικών δομών
  • Unit Tests: Αυτόματη γραφή comprehensive tests
  • Documentation: JSDoc, comments και README files
  • RegEx & Algorithms: Σύνθετα patterns και optimized algorithms
  • API Integration: HTTP requests, error handling, authentication
  • Code Explanation: Εξήγηση legacy ή πολύπλοκου κώδικα

⚠️ Τι ΔΕΝ κάνει καλά:

  • Business Logic: Δεν καταλαβαίνει τις ανάγκες του project
  • Security: Μπορεί να δημιουργήσει vulnerable κώδικα
  • Performance: Δεν optimizes για real-world constraints
  • Architecture: Δεν σχεδιάζει system-level αρχιτεκτονική

💬 Prompt Engineering Masterclass

🧠 Κλειδί: Το Copilot είναι σαν έναν talented junior developer - όσο πιο συγκεκριμένος και σαφής είσαι, τόσο καλύτερα αποτελέσματα παίρνεις!

📝 Οι 5 Χρυσοί Κανόνες του Prompting:

1️⃣ Start General → Get Specific

❌ "Write a function"

✅ "Write a function that validates user emails:
- Takes email string as parameter
- Returns {isValid: boolean, errors: string[]}
- Checks format, domain, length
- Provides Greek error messages"

2️⃣ Give Examples

"Create a date formatter:

Examples:
formatDate('2024-01-15') → '15 Ιαν 2024'
formatDate('2024-12-25') → '25 Δεκ 2024'
formatDate('invalid') → 'Άκυρη ημερομηνία'"

3️⃣ Break Complex Tasks

Αντί: "Create complete e-shop"

Κάνε:
1. "Create Product model"
2. "Create ProductService"
3. "Create ProductController"
4. "Add validation logic"

4️⃣ Avoid Ambiguity

❌ "What does this do?"
❌ "Fix this"
❌ "Make it better"

✅ "What does the calculateTax() function do?"
✅ "Fix the null pointer error in line 25"
✅ "Optimize this for better performance"

5️⃣ Set Context

"I'm building a React app with TypeScript and Tailwind CSS.

Create a responsive product card component that:
- Shows image, title, price
- Has add to cart button
- Follows accessibility best practices"

🎭 Advanced Prompting Techniques:

Role Playing Technique:
"Act as a senior React developer who prioritizes performance and accessibility.
Review this component and suggest improvements with explanations."
Constraint-Based Prompting:
"Create a sorting algorithm with these constraints:
- Don't use built-in sort() method
- Must be stable sort
- Optimize for readability over performance
- Add comments in Greek"
Step-by-Step Template:
"Explain how to implement JWT authentication:
1. Install necessary dependencies
2. Create JWT middleware
3. Implement login/logout endpoints
4. Add route protection
5. Handle token refresh"

🔥 Proven Prompt Templates:

🐛 Debugging Template

"I'm getting this error: [ERROR_MESSAGE]

In this [LANGUAGE] code:
[PASTE CODE]

Context: [WHAT YOU'RE TRYING TO DO]

Help me:
1. Fix the error
2. Explain why it happened
3. Prevent similar issues"

📚 Learning Template

"Explain [CONCEPT] in simple terms:

- What is it and why useful?
- Show basic example
- Common use cases
- Best practices
- What to avoid

Assume I know [YOUR_LEVEL] but new to this."

🔍 Code Review Template

"Review this [LANGUAGE] code for:

- Bugs and potential issues
- Performance improvements
- Best practices violations
- Security vulnerabilities
- Code readability

[PASTE CODE]

Provide specific suggestions with examples."

⚡ Quick Wins & Pro Tips:

  • Be specific about tech stack: "React with TypeScript" > "React"
  • Mention your level: "I'm a beginner" vs "I'm experienced"
  • Ask for explanations: "Also explain WHY this approach works"
  • Request alternatives: "Show me 3 different approaches"
  • Set constraints: "Without using external libraries"
  • Ask for trade-offs: "What are the pros and cons?"
