📋 Στόχος της Άσκησης

Οι σπουδαστές θα μάθουν να χειρίζονται φόρμες Flask, να κάνουν υπολογισμούς και να εμφανίζουν αποτελέσματα. Η άσκηση εστιάζει στην πρακτική εφαρμογή των βασικών αρχών web development.

🚀 Προετοιμασία Εργαστηρίου

Βήματα Εκπαιδευτή:

  1. Έλεγχος περιβάλλοντος: Βεβαιωθείτε ότι όλοι έχουν Python 3.6+
  2. Αντιγραφή αρχείων: Μοιράστε το φάκελο 03_calculator
  3. Demo εφαρμογής: Δείξτε την εφαρμογή στη διεύθυνση http://localhost:5003
  4. Επεξήγηση δομής: Παρουσιάστε τη δομή των αρχείων
⚠️ Σημαντικό: Βεβαιωθείτε ότι όλοι οι σπουδαστές μπορούν να δημιουργήσουν virtual environment πριν ξεκινήσετε. Χρησιμοποιήστε python3 αντί για python.

⏰ Προτεινόμενο Χρονοδιάγραμμα:

  • 0-10 λεπτά: Εισαγωγή και εγκατάσταση
  • 10-20 λεπτά: Ανάλυση κώδικα app.py
  • 20-25 λεπτά: Άσκηση 1 (Αλλαγή τιμών)

  • 50-60 λεπτά: Ερωτήσεις και εκτίμηση

💡 Συμβουλές Εκπαιδευτή

🎓 Πριν ξεκινήσετε:

  • Δείξτε την εφαρμογή σε λειτουργία πρώτα
  • Εξηγήστε τη δομή φακέλων (templates/, static/, app.py)
  • Τονίστε τη σημασία του virtual environment
  • Προετοιμάστε backup κώδικα για γρήγορη αποκατάσταση
  • Σημαντικό: Χρησιμοποιήστε python3 και όχι python

🔧 Κατά τη διάρκεια:

  • Ενθαρρύνετε τους σπουδαστές να δοκιμάζουν μετά από κάθε αλλαγή
  • Εξηγήστε τα error messages όταν εμφανίζονται
  • Δείξτε πως να χρησιμοποιούν το browser developer tools
  • Τονίστε τη σημασία των σχολίων στον κώδικα

🚨 Κοινά προβλήματα:

  • Virtual environment δεν ενεργοποιείται: Ελέγξτε το path και τη σύνταξη
  • "command not found: python": Χρησιμοποιήστε python3
  • Template not found: Έλεγχος ορθογραφίας και φακέλου templates/
  • CSS δεν φορτώνει: Έλεγχος φακέλου static/ και browser cache
  • Port already in use: Αλλαγή port στο app.py ή τερματισμός άλλων εφαρμογών
